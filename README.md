# Homebrew Tap

Custom Homebrew formulae by [@bogdan-calapod](https://github.com/bogdan-calapod).

## Installation

```sh
brew tap bogdan-calapod/tap
```

## Available Formulae

### pdq-cli

CLI for PDQ Connect and PDQ Detect.

```sh
brew install bogdan-calapod/tap/pdq-cli
```

Or directly without tapping first:

```sh
brew install bogdan-calapod/tap/pdq-cli
```
